This project is a simple multi-page website built using HTML, CSS, and JavaScript. It consists of three main pages 
designed to demonstrate basic front-end development concepts and user interaction.

Pages Included:

Home Page:
A visually appealing landing page that introduces the website and provides easy navigation to other sections.

Products Page:
Displays a list of products with images and descriptions.
Includes a search functionality implemented using JavaScript, allowing users to filter products in real time.

Contact Page:
Contains a registration/contact form where users can enter their details.
Built using HTML form elements and styled with CSS for a clean and responsive layout.




Technologies Used:

HTML – Structure of the web pages

CSS – Styling and layout design

JavaScript – Interactive features (search functionality, UI interactions)




Features:

Multi-page website structure

Responsive and user-friendly design

Real-time product search

Simple and clean contact/registration form

Beginner-friendly project structure




Purpose:

This project is ideal for learning and practicing front-end web development, focusing on layout design, DOM manipulation, and basic JavaScript functionality.
